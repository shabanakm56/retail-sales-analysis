# Retail Sales Performance Analysis

## Overview
Analysed 9,994 retail transactions to identify sales patterns,
profitability drivers and regional performance trends.
Built end-to-end data analytics pipeline using Python, MySQL and Power BI.

**Tools:** Python · pandas · matplotlib · seaborn · MySQL · Power BI · Jupyter Notebook

## Key Findings
- West region leads in total sales (720K+)
- Technology is most profitable category
- Furniture shows negative profit despite high sales volume
- Discounts above 40% consistently result in negative profit margins
- Sales peak every November-December — holiday season effect
- California and New York are top 2 revenue states

## Project Structure
- `01_data_cleaning.ipynb` — Data cleaning, EDA and visualizations in Python
- `superstore_clean.csv` — Cleaned dataset
- `retail_sales_dashboard.pbix` — 4-page interactive Power BI dashboard

## Steps Followed
1. Loaded and cleaned 9,994 rows using pandas
2. Fixed date columns, extracted Year and Month features
3. Built 5 charts — regional sales, category profit, monthly trend, discount vs profit, heatmap
4. Loaded data into MySQL and wrote 8 business SQL queries
5. Built 4-page Power BI dashboard with KPI cards, slicers and bar/line charts

## Dataset
Superstore Sales Dataset — Kaggle
9,994 rows · 21 columns · US retail orders 2014–2017
